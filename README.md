
## Getting Started

```bash
$ git clone git@github.com:sansroman/Library.git
$ cd Library
$ npm install
$ npm run migrate:up
$ npm run dev
```

> make sure mysql config is correct , They are at ./config/config.default.js

* Node.js >= 8.0.0 required
* MySQL required

seeder sql file at dump dir
